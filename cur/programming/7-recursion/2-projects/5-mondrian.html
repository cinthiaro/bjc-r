  4. Luego, programa el código para el caso recursivo. El bloque Mondrian ya contiene alguna estructura. Tu trabajo es averiguar qué entradas necesitará en base a cómo debería funcionar el script.

    • Ya has visto el nivel 1, un rectángulo relleno. En cada nivel, Mondrian subdivide el espacio que tiene en dos rectángulos (que se subdividirán aún más si se requieren más niveles). Por ejemplo, en el nivel 2, Mondrian dividirá aleatoriamente el área especificada por sus entradas de ancho y alto en dos piezas, y dibujará un rectángulo en cada pieza. Para el nivel 3, Mondrian  utiliza un Mondrian de nivel 2 para subdividir los dos rectángulos más pequeños (para un total de cuatro rectángulos rellenados). El nivel 4 se subdividirá de nuevo, creando ocho rectángulos, y así sucesivamente.
    • La variable  fracción contendrá una fracción aleatoria entre 1/5 y 4/5, especificando cómo subdividir el área a este nivel.
    • El algoritmo decide si subdividir la anchura o la altura.

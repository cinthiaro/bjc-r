Aquí hay una solución para el bloque de subconjuntos:

Esta solución debería recordarte el bloque pascal que has visto en Unidad 8 Laboratorio 3 Página 2: El triángulo de Pascal y la eficiencia que hace dos llamadas recursivas y añade los resultados. El bloque pascal hace dos llamadas recursivas con diferentes llamadas con diferentes entradas. Esta version de subconjuntos hace la misma llamada recursiva dos veces, una ineficiencia que puede ser corregida.

  1. Utilizando una variable de contar, como lo hiciste para el bloque pascal, cuenta cuántas llamadas recursivas se hacen para encontrar los 64 subconjuntos de una lista de seis elementos.
  2. Averigua cómo reducir el número de llamadas recursivas evitando las llamadas redundantes.
